# 🎵 Vibe Player

A modern, elegant music player for Android that plays local audio files with a beautiful Material You design.

## ✨ Features

- 🎧 **Local Music Playback** - Play music files stored on your device
- 📁 **Smart Media Scanner** - Automatic detection of audio files with filtering options
- 🎨 **Modern UI** - Built with Jetpack Compose and Material Design 3
- 💾 **Local Database** - Room database for storing track metadata
- 🏗️ **Clean Architecture** - MVVM pattern with dependency injection via Koin
- 🔄 **State Management** - Efficient state handling with Compose recomposition
- 🔍 **Filter & Search** - Filter tracks by duration and file size

## 🛠️ Tech Stack

-  **Kotlin** - Modern, concise, and safe programming language
-  **Jetpack Compose** - Declarative UI toolkit
-  **Room Database** - Local persistence layer
-  **Koin** - Dependency injection framework
-  **Android MediaStore** - Access local audio files
-  **MediaMetadataRetriever** - Extract audio metadata
-  **MVVM Architecture** - Clean separation of concerns
-  **Coroutines & Flow** - Asynchronous programming
